# UI叩き案（編集画面）

日付: 2026-05-31

## 1. 全体レイアウト（基本）

```text
[ Top Bar ]
  ├ タイトル
  ├ プロジェクト名(編集)
  ├ ユーザー/保存/エクスポート
  -------------------------------------------
[ Main Area ]
  +------------------+-----------------------------+------------------+
  |  左: メディア         |      中央: プレビュー           | 右: プロパティ   |
  |  パネル            |      クリップリスト              |  美容/色調設定 |
  |                   |                                 |  音声/速度等    |
  |  - メディア一覧     |      タイムライン(下段)          |  プロジェクト設定 |
  +------------------+-----------------------------+------------------+
  -------------------------------------------
                               | 下: ステータス/ヒストリー/ヒント |
```

## 2. 画面ごとの叩き

### A. 作品編集画面（デフォルト）
- 左
  - `素材` / `ビデオ` / `音声` / `画像` タブ
  - クリップをドラッグしてタイムラインへ
- 中央上
  - メインプレビュー
  - 再生バー、再生速度、フレーム前後移動
  - Before / After トグル
- 中央下（タイムライン）
  - トラック1: 動画
  - トラック2: BGM
  - トラック3: エフェクト
  - クリップ左端: 分割、削除、静止化
  - 右上: 自動再生 / 音量ミュート
- 右
  - `1Tap Beauty`（大きめボタン）
  - スライダー
    - 肌のなめらかさ
    - くすみ補正
    - 目の明るさ
    - 歯の明るさ
    - 自動補正
  - プリセット: ナチュラル / ソフト / グラマラス
  - テキスト、字幕、BGM音量

### B. テーマ切替メニュー
- `見た目` ボタン（右上）
  - `かわいい`（pink）
  - `クリーン`（cool）
  - `作業モード（無音/最小アニメ）`
- テーマ適用は即時反映
- 各テーマは独立プリセットを保存

## 3. テーマ別「叩き」見え方

### 3-1 可愛いテーマ（cute）
- 背景: ピンク系パステル
- パネル: 白+淡いピンク、角丸大きめ
- コントラール: 明るいピンク・ラベンダー
- アニメ: 軽いフェード＋小さな揺れ
- ボタン: 影＋やや立体
- 文字: 親しみやすい丸みのある書体

### 3-2 男性向けテーマ（cool）
- 背景: ダーク/ネイビー
- パネル: 濃いグレー、角丸小さめ
- コントロール: コントラスト強め、影は控えめ
- アニメ: フェードのみ
- ボタン: 線的で視線誘導が明確
- 文字: 角がはっきりしたゴシック系

## 4. 主要導線（3クリック以内）
1. 素材を左パネルへドロップ  
2. 中央タイムラインに配置  
3. 右の `1Tap Beauty` で即時確認 → `書き出し` で保存

## 5. UIコンポーネントサイズ
- ボタン最小
  - PC: 高さ 44 / 52
  - タブレット: 48 / 56
- スライダー高さ: 32
- タイムラインクリップ高さ: 28

## 6. 画面遷移（最小）
- 新規作成 → 編集画面
- 編集画面 → 書き出し設定（サイドシート）
- 書き出し中 → 進捗オーバーレイ（キャンセル可能）
- 終了 → プロジェクト保存を確認

## 7. 叩き段階で決めるべきこと
- 最優先
  - 2テーマ初期値の固定
  - 左右パネルの幅比率（40/30/30 など）
  - `1Tap Beauty` のラベル文言
  - テーマ切替トリガー位置
- 次で決める
  - アイコンセット
  - 角丸・影の厳密数値
  - タイムライン色分けルール
